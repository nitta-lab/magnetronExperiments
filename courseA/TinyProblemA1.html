小課題A1

6つの小課題は，いずれも A，B，C，D，E の5つのクラスで構成され， A クラスの m() メソッドが呼び出されるところから始まって， 以下のように，E クラスのあるインスタンスが，C クラスのあるインスタンスをフィールド c で参照する (E クラスのフィールド c に C クラスのインスタンスへの参照が代入される)ところで終了します．
小課題間でA~Eのクラスの名称は共通ですが，各クラス内に定義されているメソッドやメソッド間の呼び出し関係はそれぞれ異なります．
なお，Main クラスはすべての小課題で共通なので省略します．必要に応じてこちらをご参照ください．

:
public class E {
    C c = null;
    :
    public void someMethod(...) {
        :
        this.c = Cのインスタンスへの参照;
    }
    :
}
